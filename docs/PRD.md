# 產品需求文件 (PRD) - 心情選歌系統

## 1. 專案概述

- **背景與動機**：現代人生活忙碌，音樂是紓壓與調節情緒的重要媒介，但面對龐大的歌曲庫時，常常不知道要聽什麼。本系統希望透過「心情」與「天氣」兩個直覺化的維度，幫助使用者快速找到一首符合當下情境的歌曲，省去漫無目的瀏覽歌單的時間。同時也提供「他人推薦」功能，讓使用者能發現更多意想不到的好歌。
- **目標用戶**：
  - **一般使用者**：所有喜歡聽音樂的人，以年輕族群（18–35 歲）為主要客群，但不限年齡。
  - **管理員**：負責維護歌曲資料庫、管理推薦內容與系統設定的角色。
- **核心價值主張**：提供一個簡單直覺的互動介面，讓使用者只需點選心情或天氣，即可獲得一首隨機推薦歌曲，降低「選歌障礙」，讓音樂無縫融入日常生活。

## 2. 功能需求

系統涵蓋以下核心功能，各項功能的使用者故事 (User Story) 如下：

1. **心情選擇按鈕**
   - 作為 **一般使用者**，我希望 **能夠在首頁點選代表我當下心情的按鈕（如：開心、悲傷、放鬆、焦慮、興奮等）**，以便 **系統能根據我的心情推薦一首適合的歌曲**。

2. **天氣選擇按鈕**
   - 作為 **一般使用者**，我希望 **能夠在首頁點選代表當天天氣的按鈕（如：晴天、雨天、陰天、下雪等）**，以便 **系統能根據天氣氛圍推薦一首符合情境的歌曲**。

3. **他人推薦曲目按鈕**
   - 作為 **一般使用者**，我希望 **能夠點選「他人推薦」按鈕，瀏覽其他使用者推薦的歌曲清單**，以便 **發現自己平常不會主動搜尋的好歌，拓展音樂視野**。

4. **選歌結果頁面**
   - 作為 **一般使用者**，我希望 **在選擇心情或天氣後，能看到一個清楚的結果頁面，顯示推薦歌曲的名稱、歌手、歌曲風格，以及該歌曲的試聽連結或嵌入播放器**，以便 **我能立即了解並試聽這首歌**。

5. **重新選擇按鈕**
   - 作為 **一般使用者**，我希望 **在結果頁面能點選「重新選擇」按鈕回到首頁，或點選「換一首」按鈕重新隨機推薦**，以便 **在不滿意推薦結果時能快速獲得新的建議，而不必手動返回**。

6. **歌曲管理（後台）**
   - 作為 **管理員**，我希望 **能夠新增、編輯、刪除歌曲資料（包含歌名、歌手、風格、對應心情標籤、對應天氣標籤、試聽連結）**，以便 **持續豐富與維護歌曲資料庫的內容品質**。

7. **使用者推薦歌曲**
   - 作為 **一般使用者**，我希望 **能夠提交我想推薦的歌曲（包含歌名、歌手、推薦原因）**，以便 **與其他使用者分享我喜歡的音樂**。

## 3. 非功能需求

- **技術限制**：
  - 框架與後端：基於 Web 服務，後端須使用 Python `Flask` 框架。
  - 前端與樣式：HTML 模板使用 `Jinja2` 渲染，搭配 Vanilla CSS 與 JavaScript。
  - 資料庫：採用 `SQLite` 關聯式資料庫。
- **效能與安全考量**：
  - **安全性**：管理員登入須實作密碼雜湊加密機制（例如 bcrypt），避免明碼儲存；針對表單輸入須防範 SQL Injection 及 XSS 攻擊。使用者推薦歌曲功能須對輸入內容做適當的驗證與過濾。
  - **效能**：歌曲隨機推薦查詢應在合理時間內回應（< 1 秒）。頁面載入需快速流暢，確保良好的使用者體驗。
  - **可用性**：介面設計須簡潔直覺，支援行動裝置的響應式設計 (RWD)，確保在手機與平板上也能順暢操作。

## 4. MVP 範圍 (Minimum Viable Product)

- **Must Have (必須有)**：
  - 首頁顯示心情選擇按鈕（至少 5 種心情）。
  - 首頁顯示天氣選擇按鈕（至少 4 種天氣）。
  - 點選心情或天氣後，隨機推薦一首對應的歌曲。
  - 選歌結果頁面，顯示歌曲名稱、歌手與風格資訊。
  - 「重新選擇」按鈕回到首頁，「換一首」按鈕重新隨機推薦。
  - 基礎歌曲資料庫，預先填入足夠數量的歌曲資料（每個心情 / 天氣至少 5 首）。

- **Should Have (應該有)**：
  - 他人推薦曲目按鈕與推薦清單瀏覽頁面。
  - 使用者提交推薦歌曲的表單功能。
  - 管理員登入與歌曲 CRUD 後台管理介面。
  - 歌曲試聽連結（如 YouTube 外部連結）。

- **Nice to Have (加分項)**：
  - 嵌入式音樂播放器（如 YouTube / Spotify embed）。
  - 「心情 + 天氣」組合篩選推薦。
  - 推薦歌曲的按讚（❤️）功能。
  - 推薦歷史紀錄頁面，查看過去被推薦過的歌曲。
  - 動態背景效果（根據選擇的心情 / 天氣改變頁面氛圍）。
